================================================================================
 INSIGHTS QUICK REFERENCE
================================================================================

AVAILABLE METRICS FIELDS:
- impressions: Number of times ad was displayed
- clicks: Number of clicks on ad
- spend: Amount spent in account currency
- reach: Number of unique users reached
- frequency: Average number of times ad shown to each person
- ctr: Click-through rate (%)
- cpc: Cost per click
- cpm: Cost per thousand impressions
- cpp: Cost per person
- actions: Conversion actions
- action_values: Monetary value of conversions
- conversions: Total conversions
- conversion_value: Total value of conversions

AVAILABLE BREAKDOWNS:
Demographics:
  - age: User age brackets
  - gender: User gender
  - country: Country of user
  - region: State/province of user
  - city: City of user

Device & Placement:
  - device: Device type (mobile, desktop, etc.)
  - placement: Where ad was shown (feed, right_column, etc.)
  - platform: Platform (facebook, instagram, audience_network, etc.)

Conversion:
  - conversion_device: Device at time of conversion
  - conversion_destination: Where conversion happened

Other:
  - audience_id: Custom audience ID
  - frequency_value: Frequency bracket
  - impression_device: Device when ad was shown

DATE PRESETS - ALL OPTIONS:
- last_7d: Last 7 days
- last_14d: Last 14 days
- last_28d: Last 28 days
- last_30d: Last 30 days (default)
- last_90d: Last 90 days
- today: Today only
- yesterday: Yesterday only
- this_week: Current week
- last_week: Previous week
- last_month: Previous month
- this_quarter: Current quarter
- last_3m: Last 3 months
- lifetime: All time data

================================================================================
 LEAD FORM OPERATIONS
================================================================================

Lead forms are used with Lead Generation campaigns to collect user information
directly on Facebook/Instagram without users leaving the platform.

IMPORTANT: Lead forms are created on Facebook PAGES, not Ad Accounts.
You must provide a "page_id" for create and list operations.

================================================================================
 CREATE LEAD FORM
================================================================================

Basic lead form with standard questions:
{
  "ad_account_id": "120244256006770196",
  "action": "create_lead_form",
  "page_id": "864501833422699",
  "lead_form": {
    "name": "Contact Us Form",
    "locale": "en_US",
    "questions": [
      {"type": "EMAIL"},
      {"type": "PHONE"},
      {"type": "FULL_NAME"}
    ],
    "privacy_policy": {
      "url": "https://example.com/privacy"
    },
    "follow_up_action_url": "https://example.com/thank-you"
  }
}

Lead form with custom questions and intro screen:
{
  "ad_account_id": "120244256006770196",
  "action": "create_lead_form",
  "page_id": "864501833422699",
  "lead_form": {
    "name": "Product Interest Form",
    "locale": "en_US",
    "questions": [
      {"type": "EMAIL"},
      {"type": "PHONE"},
      {"type": "FULL_NAME"},
      {
        "type": "CUSTOM",
        "key": "service_interest",
        "label": "Which service are you interested in?",
        "options": [
          {"value": "Web Development", "key": "web"},
          {"value": "Mobile App", "key": "mobile"},
          {"value": "Digital Marketing", "key": "marketing"}
        ]
      },
      {
        "type": "CUSTOM",
        "key": "budget_range",
        "label": "What is your budget range?",
        "options": [
          {"value": "Under $5,000", "key": "low"},
          {"value": "$5,000 - $10,000", "key": "medium"},
          {"value": "Over $10,000", "key": "high"}
        ]
      }
    ],
    "privacy_policy": {
      "url": "https://example.com/privacy",
      "link_text": "Privacy Policy"
    },
    "intro": {
      "title": "Get a Free Consultation",
      "description": "Fill out this form and our team will contact you within 24 hours."
    },
    "thank_you_page": {
      "title": "Thank You!",
      "body": "We've received your information and will be in touch soon.",
      "button_text": "Visit Our Website",
      "button_url": "https://example.com"
    },
    "follow_up_action_url": "https://example.com/thank-you"
  }
}

Available Standard Question Types:
- EMAIL: Email address (auto-filled from profile)
- PHONE: Phone number (auto-filled from profile)
- FULL_NAME: Full name (auto-filled from profile)
- FIRST_NAME: First name only
- LAST_NAME: Last name only
- CITY: City
- STATE: State/Province
- COUNTRY: Country
- ZIP: Zip/Postal code
- DOB: Date of birth
- GENDER: Gender
- JOB_TITLE: Job title
- COMPANY_NAME: Company name
- CUSTOM: Custom question (requires label, optional options for multiple choice)
